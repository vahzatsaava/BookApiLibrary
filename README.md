# BookApiLibrary


Как запустить проект

1. Запустите Docker Compose

```bash
docker-compose up -d
``
либо используя интерфейс

2. Перейдите на контроллер

http://localhost:8086/books


3. Перейдите на Swagger UI (REST-интерфейс)

http://localhost:8086/swagger-ui/index.html#/


 4. Зарегистрируйтесь

Перейдите в раздел регистрации в Swagger и зарегистрируйте нового пользователя.  
Можно использовать дефолтные значения которые дает свагер


 Технологии

- Java 17
- Spring Boot
- Junit/Mockito
- PostgreSQL
- Docker & Docker Compose
- Swagger / OpenAPI
